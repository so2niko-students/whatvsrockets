# MVC
### Model View Controller

## Model
Логика, бизнес-логика, работа с БД, отправка запросов, получение ответов. 

## View
Отображение. Рендер данных, перерендер, управление DOM элементами, работа с _DOM Events_

## Controller
Посредник между Model & View.
